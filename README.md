## 📌 Sobre o Projeto

Este é o frontend do site institucional da **Protonfit Equipamentos**, desenvolvido com foco em performance, responsividade e uma experiência moderna para o usuário.

A aplicação permite que clientes conheçam a empresa, explorem o catálogo de produtos e entrem em contato para orçamentos.
